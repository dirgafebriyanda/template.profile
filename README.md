# Template Profile

A modern Profile UI template built with HTML and Tailwind CSS.

## Preview

Live Demo: https://template-profile-five.vercel.app/

## Features

* Profile form
* Responsive design
* Tailwind CSS
* Clean and modern UI
* Easy customization

## Getting Started

### Clone Repository

```bash
git clone https://github.com/dirgafebriyanda/template.profile.git
cd template.profile
npm install
```

### Open Project

Open `index.html` directly in your browser.

Or run a local server:

```bash
npm run dev
```

## Customization

You can easily customize:

* Colors
* Fonts
* Form fields
* Animations
* Layout
